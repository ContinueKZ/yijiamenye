# 益嘉门业官网

益嘉门业官网是一个基于 Vue + Vite 的门业展示型网站，主打简约、高级、图片优先的产品展示体验。

网站不支持在线支付、购物车或下单流程，核心目标是让客户通过产品图片、分类浏览、产品描述和联系方式产生咨询意向。

## 技术栈

- Vue 3
- Vite
- Vue Router
- lucide-vue-next
- CSS 响应式布局

## 页面

- 首页
- 产品中心
- 产品详情
- 关于我们
- 联系我们

## 本地运行

```bash
npm install
npm run dev
```

默认访问地址：

```txt
http://localhost:5173/
```

## 构建

```bash
npm run build
```

## 内容维护

站点信息、联系方式、产品分类、产品数据和图片地址集中维护在：

```txt
src/constants/site.js
```

后续修改联系方式、产品分类或产品图片时，优先修改这个文件，避免内容散落在多个页面中。

## 设计说明

设计系统文档位于：

```txt
DESIGN.md
```

整体风格遵循简约、高级、展厅感、图片优先的方向，并兼容 PC、平板和手机三端。
